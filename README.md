# Crypto Course Tests Hub

Статический репозиторий для публикации тестов курса **BlockCapital Crypto Summit** на GitHub Pages.

Что входит:
- статическая платформа из папки `Платформа крипто-курса`;
- верифицированные тесты занятий в `Платформа крипто-курса/data/lessons/lesson-NN.json`;
- GitHub Actions workflow для автоматической публикации на GitHub Pages.

Основная рабочая инструкция — в `Платформа крипто-курса/README.md`.

## Чем отличается от ai-course-tests-hub

В AI-хабе данные парсились из markdown-тестов жёстким парсером. Здесь готовые тесты
пайплайна выходят в разном формате от занятия к занятию, поэтому источник истины —
**верифицированные вручную JSON-файлы** `data/lessons/lesson-NN.json` (по одному на занятие).
Сборщик их только валидирует и склеивает в `data/course-data.json`. Сам фронтенд (index.html,
app/main.js, app/styles.css) переиспользован из AI-хаба и формат-агностичен.
